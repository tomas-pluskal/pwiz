ProteoWizard Release Information

Latest Release

The latest stable branch of ProteoWizard is 1.5.x. The latest stable release/package is 1.5.2.

Packages are avaliable at the ProteoWizard download page:
http://proteowizard.sourceforge.net/download.html

Data formats supported in 1.5.2 (note: vendor formats require Windows with vendor libraries):

  • mzML 1.1
  • Thermo RAW
  • Waters RAW
  • Bruker FID/YEP/BAF
  • mzXML
  • MGF

ProteoWizard Versioning

ProteoWizard versions are labeled x.y.z:

  • x: major version (major additions or API changes)
  • y: minor version (new features)
  • z: patch version (bug and cosmetic fixes)
