Proprietary Formats

ProteoWizard depends on vendor-supplied software libraries to access data in proprietary formats.

Common requirements:

  • Windows XP or Vista
  • MSVC 2008 SP1 x86 redistributable installed

Vendor-specific requirements:

Vendor Formats Vendor Required Software Required Runtime Libraries
Agilent MassHunter .d distributed with ProteoWizard .NET 2.0 or higher
Applied Biosystems WIFF Protein Pilot 3 free trial .NET 3.5 SP1
Bruker Compass .d, YEP, BAF, FID distributed with ProteoWizard MSVC 2005 SP1 x86 redistributable; .NET 2.0 or higher
Thermo Fisher RAW MSFileReader free download none
Waters MassLynx .raw MassLynx none
