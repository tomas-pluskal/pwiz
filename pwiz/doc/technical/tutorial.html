ProteoWizard Tools Tutorial

Downloading the tools

Zip files containing binary executables for various platforms may be downloaded from the ProteoWizard home page.

File format conversion

msconvert is a command line tool for converting between various file formats.

Currently supported formats:

  • Read: mzML, mzXML, Thermo RAW, MGF
  • Write: mzML, mzXML

Vendor format reading restrictions:

  • Thermo RAW: Windows with XCalibur XDK installed

Examples:

  • msconvert data.RAW
    (creates data.mzML in current directory)

  • msconvert data.RAW --mzXML
    (creates data.mzXML in current directory)

  • msconvert *.RAW -o my_output_dir
    (converts all files matching *.RAW to mzML, output files created in my_output_dir)

  • msconvert data.RAW --zlib --centroid [2,3]
    (use zlib compression for binary data arrays, and centroid for msLevels in [2,3])

Data extraction

msaccess is a command line tool for extracting data and metadata from data files.

Examples:

  • msaccess data.mzML -x spectrum_table
    (creates data.mzML.spectrum_table.txt with summary information for all spectra)

  • msaccess data.mzML -x "binary 0-3"
    (creates files data.mzML.binary.*.txt with binary data for spectra 0 through 3)

  • msaccess data.mzML -x image
    (creates data.mzML.image* with pseudo-2D-gel image of the data file)
