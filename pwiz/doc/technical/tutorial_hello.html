hello_pwiz Tutorial

This tutorial will take you through the steps to create a new command-line tool using ProteoWizard.

  1. Download and build ProteoWizard. In what follows, <pwiz_root> refers to the directory where you put ProteoWizard.

  2. Save the file hello_pwiz.cpp and put it in <pwiz_root>/pwiz_tools/examples.

  3. Edit <pwiz_root>/pwiz_tools/examples/Jamfile.jam, adding the following two code blocks:

    exe hello_pwiz
    : hello_pwiz.cpp
      ../../pwiz/data/msdata//pwiz_data_msdata
    : <include>../..
    ;

    install hello_pwiz_install
    : hello_pwiz
    : <location>../.. # this puts the executable in <pwiz_root>
    ;

  4. Build hello_pwiz from <pwiz_root>:
    $ quickbuild.sh pwiz_tools/examples//hello_pwiz_install

  5. You should now have an executable "hello_pwiz" in <pwiz_root> You can test it:
    $ hello_pwiz example_data/tiny.pwiz.1.1.mzML

Now that you're up and running, you'll probably want to change the location where the exectuable is installed, put your source files in their own directory, and make adjustments for your personal build environment.
